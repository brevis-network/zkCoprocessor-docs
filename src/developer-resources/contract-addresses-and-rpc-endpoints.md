## SDK&#x20;

Repo: [https://github.com/brevis-network/brevis-sdk](https://github.com/brevis-network/brevis-sdk)

> **Note:** 
Please make sure SDK version is _<mark style="color:red;">**no less than**</mark>_ 0.3.17. It is not backward-compatible.

To update SDK version, go to the file folder  which contains go.mod file in terminal and run the following command

`go get github.com/brevis-network/brevis-sdk@latest`
